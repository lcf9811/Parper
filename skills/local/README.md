# 本地 Skill 文件夹

此文件夹用于存放自定义 Skill 文件。

## Skill 文件格式

每个 Skill 应该是一个 `.md` 文件，格式如下：

```markdown
# Skill Name

## 名称
code_reviewer

## 显示名称
代码审查助手

## 描述
专业的代码审查专家，帮助发现代码中的问题和改进点

## 系统提示词
你是一个资深的代码审查专家。你的职责是：
1. 检查代码中的潜在 bug 和安全漏洞
2. 评估代码的可读性和可维护性
3. 提出具体的改进建议
4. 遵循最佳实践和编码规范

请用中文回答，并给出明确的建议。
```

## 字段说明

- **名称**: Skill 的唯一标识（英文，用于程序识别）
- **显示名称**: 在界面上显示的名称
- **描述**: Skill 的简短描述
- **系统提示词**: 发送给 AI 的系统提示词

## 自动加载

系统启动时会自动读取此文件夹下的所有 `.md` 文件，并加载到数据库中。
通过 API `POST /api/skills/reload` 可以手动刷新。
